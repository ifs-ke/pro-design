# **App Name**: DesignCost Pro

## Core Features:

- Cost Input: Input fields for material costs, labor hours, and internal operational costs.
- Tax Calculation: Automated calculations for tax based on a configurable rate.
- Margin Control: Configurable percentage margins for profit calculation, with a redline at 18%.
- Budget Allocation: Allocation of percentage towards salaries, savings, future developments, and CSR with customizable fields and distribution percentages. Provide visual feedback when all the percentages add to 100%.
- Material Suggestion: AI powered "tool" that provides dynamic suggestions for material alternatives based on budget constraints, analyzing price and availability, and offering recommendations with pros and cons.
- Display Output: Display the pricing calculation results in an organised, easy-to-read manner.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of creativity and sophistication.
- Background color: Light grey (#ECEFF1) to provide a neutral backdrop.
- Accent color: Teal (#009688) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font for a clean and modern look.
- Use simple, line-based icons to represent different cost categories.
- Use a grid-based layout to organize information clearly.
